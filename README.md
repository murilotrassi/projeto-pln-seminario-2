# projeto-pln-seminario-2
Contém os códigos dos detectores e explicadores de fake news. Para rodar no seu computador, é necessário colocar as bases do Fake.BR Corpus em alugm local do seu computador e passar esse caminho no código, na 5ª e  6ª células de código. A base Fake.BR Corpus pode ser encontrada aqui: https://github.com/roneysco/Fake.br-Corpus
